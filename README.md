## xum - cross uplink multiplexer

### Installation
    $ sudo npm xum (TODO: publish)
    $ sudo npm install -g git://github.com/hakt0r/xum.git

### Usage:
    $ sudo xum connect user@host:port
    $ sudo xum add ppp0 --cheap --limit=5k
    $ sudo xum del ppp0
